# ESPHome Radio Analyzer

Projet ESPHome pour transformer une LilyGO LoRa32 SX1278 433 MHz en analyseur radio.

## Matériel

- LilyGO LoRa32 T3 V1.6.1
- SX1278 433 MHz
- OLED SSD1306 128x64

## Objectif

- Mesure RSSI
- Détection d'activité radio
- Analyse bande 433 MHz
- Intégration Home Assistant
